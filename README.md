## Project Overview

The Multi-Label Text Categorization System is a Machine Learning and Natural Language Processing (NLP) project designed to automatically classify text documents into multiple categories. Unlike traditional text classification, where a document belongs to only one category, this system can assign multiple relevant labels to a single document.

The project uses TF-IDF Vectorization and Logistic Regression with a One-vs-Rest classification strategy to perform multi-label classification.

## Machine Learning Techniques Used

### TF-IDF Vectorization

TF-IDF converts textual content into numerical feature vectors that can be processed by machine learning algorithms.

### Logistic Regression

Logistic Regression is used as the base classifier for category prediction.

### One-vs-Rest Classifier

The One-vs-Rest strategy enables multi-label classification by training one classifier for each category.
